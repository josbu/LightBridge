# 用户支付页面（需登录 + 启用支付）

本章介绍充值、订阅购买和订单管理功能。

---

## 充值购买 `/purchase`

充值和订阅购买页面。

### 充值流程

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 选择充值金额 | 点击快捷金额按钮或输入自定义金额 | 快捷金额：10 / 20 / 50 / 100 / 200 / 500 / 1000 / 2000 / 5000 |
| 选择支付方式 | 点击支付方式按钮 | 支持支付宝、微信支付、Stripe、Airwallex |
| 查看手续费 | 查看支付方式旁显示 | 费率 > 0 时显示 |
| 确认支付 | 点击确认按钮 | 创建订单并跳转到支付流程 |

### 订阅流程

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 选择订阅计划 | 点击计划卡片 | 展示计划详情、价格、配额、功能列表 |
| 选择支付方式 | 点击支付方式按钮 | 同上 |
| 确认支付 | 点击确认按钮 | 创建订单并跳转到支付流程 |

### 支付等待

创建订单后，系统根据支付方式自动选择等待模式：

| 支付方式 | 等待模式 | 说明 |
|---------|---------|------|
| 支付宝/微信 QR 码 | QR 码等待 | 显示二维码，手机扫码支付，每 3 秒轮询结果 |
| Stripe Card/Link | Stripe Payment Element | 页面内直接确认支付 |
| Airwallex | 跳转到 Airwallex 收银台 | 跳板页自动重定向 |
| 微信 OAuth | 微信授权页 | 授权后回调恢复支付 |
| 微信 JSAPI | WeixinJSBridge | 微信内调起支付 |

## 我的订单 `/orders`

查看和管理所有订单。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 按状态筛选 | 选择状态下拉 | All / PENDING / COMPLETED / FAILED / REFUNDED |
| 取消订单 | 点击取消按钮 | 仅 PENDING 状态可用，弹出确认对话框 |
| 申请退款 | 点击退款按钮 | 仅 COMPLETED + 退款资格提供者可用，输入退款原因后提交 |
| 刷新列表 | 点击刷新按钮 | 重新加载订单列表 |
| 跳转充值 | 点击「返回充值」按钮 | 跳转到 `/purchase` |
| 翻页浏览 | 使用分页控件 | 查看历史订单 |

---

## 支付结果 `/payment/result`

支付完成后显示结果。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看支付结果 | 自动显示 | 显示成功 / 失败 / 处理中状态 |
| 返回充值页 | 点击返回按钮 | 跳转到充值页面 |

---

## QR 码支付 `/payment/qrcode`

QR 码支付等待页面。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 显示支付二维码 | 自动显示 | Canvas 渲染 + 品牌 logo 叠加 |
| 扫码支付 | 使用手机扫码 | 支持支付宝/微信扫码 |
| 查看倒计时 | 自动显示 | QR 码有效时间倒计时 |
| 取消支付 | 点击取消按钮 | 取消当前订单 |
| 查看支付结果 | 自动轮询 | 每 3 秒轮询订单状态 |

---

## Stripe 支付 `/payment/stripe`

Stripe Payment Element 支付页面。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 选择支付方式 | 点击 Stripe 支持的方式 | 支持 Card / Alipay / WeChat Pay / Link |
| 确认支付 | 填写支付信息并确认 | 直接在页面内完成支付 |

---

## Airwallex 支付 `/payment/airwallex`

Airwallex 国际支付跳板页面。自动初始化 SDK 后跳转到 Airwallex 收银台，加载失败时显示错误提示。

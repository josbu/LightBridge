# 管理员 - 支付与订单

本章介绍支付仪表盘、订单管理和订阅计划管理功能。

---

## 支付仪表盘 `/admin/orders/dashboard`

支付系统全局概览。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看支付统计 | 自动加载 | 收入、订单量、成功率等指标 |
| 查看趋势图表 | 自动加载 | 收入和订单趋势 |
| 管理支付提供商 | 点击提供商卡片 | 创建/编辑/启用/禁用提供商 |

### 支付提供商管理

| 操作 | 说明 |
|------|------|
| 创建提供商 | 选择类型（EasyPay / Alipay / WeChat / Stripe / Airwallex） |
| 编辑提供商 | 修改配置参数 |
| 启用/禁用 | 开关切换 |
| 退款开关 | 控制是否允许退款 |
| 用户自助退款 | 控制用户是否可自行申请退款 |
| 拖拽排序 | 调整提供商优先级 |

### 支付提供商配置字段

| 提供商 | 支持的支付类型 | 支付模式 | Webhook 路径 |
|--------|---------------|---------|-------------|
| EasyPay | alipay, wxpay | qrcode, popup | `/api/v1/payment/webhook/easypay` |
| Alipay | alipay | redirect / precreate | `/api/v1/payment/webhook/alipay` |
| WeChat | wxpay | — | `/api/v1/payment/webhook/wxpay` |
| Stripe | card, alipay, wxpay, link | — | `/api/v1/payment/webhook/stripe` |
| Airwallex | airwallex | — | `/api/v1/payment/webhook/airwallex` |

每种支付类型的限额配置：单笔最小/最大、日限额。

---

## 订单管理 `/admin/orders`

管理所有支付订单。

### 搜索与筛选

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索订单 | 输入搜索关键词 | 按订单号搜索 |
| 按状态筛选 | 选择状态下拉 | All / PENDING / COMPLETED / FAILED / REFUNDED |
| 按支付方式筛选 | 选择方式下拉 | 按支付方式过滤 |
| 按用户筛选 | 输入用户 ID | 按用户过滤 |
| 按时间范围筛选 | 选择起止时间 | 按创建时间过滤 |

### 订单操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看订单详情 | 点击订单行 | 展开订单详细信息 |
| 处理退款 | 点击退款按钮 | 为符合条件的订单退款 |
| 导出订单 | 点击导出按钮 | 导出为 CSV / Excel |

### 订单状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| PENDING | 黄色 | 待支付 |
| PAID | 蓝色 | 已支付 |
| RECHARGING | 蓝色 | 充值中 |
| COMPLETED | 绿色 | 已完成 |
| EXPIRED | 灰色 | 已过期 |
| CANCELLED | 灰色 | 已取消 |
| FAILED | 红色 | 失败 |
| REFUND_REQUESTED | 橙色 | 退款申请中 |
| REFUNDING | 橙色 | 退款中 |
| REFUNDED | 紫色 | 已退款 |

## 订阅计划管理 `/admin/orders/plans`

管理订阅计划。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看计划列表 | 自动加载 | 展示所有订阅计划 |
| 创建计划 | 点击创建按钮 | 打开创建表单 |
| 编辑计划 | 点击编辑按钮 | 打开编辑表单 |
| 删除计划 | 点击删除按钮 | 二次确认后删除 |
| 启用/禁用计划 | 点击状态切换 | 切换计划可用状态 |

### 订阅计划字段

| 字段 | 说明 |
|------|------|
| 计划名称 | 订阅计划名称 |
| 平台 | 所属平台（Claude / Gemini / Antigravity） |
| 价格 | 订阅价格（支持原价删除线和折扣百分比） |
| 有效期 | 按天 / 月 / 年 |
| 配额 | 速率倍率、日 / 周 / 月限额 |
| 模型范围 | 支持的模型列表 |
| 功能列表 | 计划包含的功能 |

# 管理员 - 联盟推广

本章介绍联盟推广的邀请记录、返利记录和转账记录管理。

---

## 邀请记录 `/admin/affiliates/invites`

查看所有用户的邀请记录。

### 搜索与筛选

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索记录 | 输入搜索关键词 | 模糊搜索（300ms 防抖） |
| 按时间筛选 | 选择起止日期 | 按邀请时间过滤 |
| 排序 | 点击列头 | 支持服务端排序 |
| 翻页 | 使用分页控件 | 查看更多记录 |

### 表格列说明

| 列名 | 说明 |
|------|------|
| 邀请人 | ID / 邮箱 / 用户名 |
| 被邀请人 | ID / 邮箱 / 用户名 |
| 邀请码 | 使用的邀请码 |
| 累计返利 | 该邀请产生的累计返利金额 |
| 邀请时间 | 注册时间 |

### 用户概览弹窗

点击用户邮箱弹出联盟用户概览，显示：邀请码、返利比例、已邀请人数、已产生返利的被邀请人数、可用配额、历史配额。

---

## 返利记录 `/admin/affiliates/rebates`

查看所有返利记录。

### 搜索与筛选

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索记录 | 输入搜索关键词 | 模糊搜索 |
| 按时间筛选 | 选择起止日期 | 按返利时间过滤 |
| 查看用户概览 | 点击用户邮箱 | 弹出联盟用户概览弹窗 |

### 表格列说明

| 列名 | 说明 |
|------|------|
| 订单 | ID + 外部交易号 |
| 邀请人 | 发起邀请的用户 |
| 被邀请人 | 被邀请的用户 |
| 订单金额 | 订单总金额（美元） |
| 实际支付 | 实际支付金额（人民币） |
| 返利金额 | 返利金额（绿色高亮） |
| 支付方式 | 使用的支付方式 |
| 订单状态 | 彩色标签显示状态 |
| 返利时间 | 返利发生时间 |

---

## 转账记录 `/admin/affiliates/transfers`

查看所有联盟余额转账记录。

### 搜索与筛选

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索记录 | 输入搜索关键词 | 模糊搜索 |
| 按时间筛选 | 选择起止日期 | 按转账时间过滤 |

### 表格列说明

|------|------|

# 管理员 - 核心管理

本章介绍管理员的核心管理功能：仪表盘、用户管理和分组管理。

---

## 管理仪表盘 `/admin/dashboard`

管理员主仪表盘，展示系统全局指标。

| 功能区 | 功能 | 操作方式 |
|--------|------|---------|
| 核心指标卡片 | 查看 API 密钥数、服务账号数、今日请求数、今日新增用户 | 自动加载 |
| Token 统计卡片 | 查看今日/总计 Token 数、三类费用（实际/账号/标准） | 自动加载 |
| 性能指标 | 查看 RPM/TPM、平均响应时间、活跃用户数 | 自动加载 |
| 模型分布图表 | 查看各模型使用分布 | 点击排名项可跳转到用户用量详情 |
| Token 使用趋势 | 查看 Token 使用趋势折线图 | 自动加载 |
| 用户使用趋势 | 查看 Top 12 用户的使用趋势 | 自动加载 |
| 刷新数据 | 点击刷新按钮 | 重新加载所有数据 |

---

## 用户管理 `/admin/users`

用户全生命周期管理。

### 搜索与筛选

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索用户 | 输入搜索关键词 | 模糊搜索（300ms 防抖） |
| 按角色筛选 | 选择角色下拉 | All / Admin / User |
| 按状态筛选 | 选择状态下拉 | All / Active / Disabled |
| 按分组筛选 | 选择分组下拉 | 按专属分组过滤 |
| 动态属性筛选 | 使用属性过滤器 | 按自定义用户属性筛选 |
| 列设置 | 点击列设置按钮 | 切换列的显隐，持久化到 localStorage |
| 属性配置 | 点击属性配置按钮 | 管理自定义用户属性定义 |
| 排序 | 点击列头排序 | 支持服务端排序 |

### 用户操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 创建用户 | 点击「创建用户」按钮 | 打开创建弹窗 |
| 编辑用户 | 点击编辑按钮 | 打开编辑弹窗 |
| 删除用户 | 点击删除按钮 | 二次确认后删除（管理员不可删除） |
| 启用/禁用用户 | 点击状态切换 | 切换用户状态（管理员不可操作） |
| 充值/扣款 | 点击余额列或操作菜单 | 为用户充值或扣款 |
| 查看余额历史 | 点击余额链接 | 打开余额历史弹窗 |
| 管理 API 密钥 | 点击操作菜单 → 查看密钥 | 打开用户密钥列表弹窗 |
| 管理分组分配 | 点击操作菜单 → 允许分组 | 管理用户可用分组 |
| 设置平台配额 | 点击操作菜单 → 平台配额 | 设置各平台的使用配额 |
| 替换分组 | 点击分组列 → 选择新分组 | 替换用户的专属分组 |
| 查看 API 密钥归属 | 点击密钥查找按钮 | 粘贴密钥查看归属用户 |

### 批量操作

选中多行后，底部出现批量操作栏：

| 功能 | 说明 |
|------|------|
| 批量编辑 | 批量更新选中用户的角色、状态、分组、余额等 |
| 批量删除 | 批量删除选中用户 |

## 分组管理 `/admin/groups`

API 分组（Group）的创建、配置和管理。

### 列表操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索分组 | 输入搜索关键词 | 模糊搜索 |
| 按协议筛选 | 选择上游协议下拉 | 按 claude / openai / gemini 等筛选 |
| 按状态筛选 | 选择状态下拉 | active / inactive |
| 按独占性筛选 | 选择独占筛选 | exclusive / public |
| 创建分组 | 点击「创建分组」按钮 | 打开创建表单 |
| 编辑分组 | 点击编辑按钮 | 打开编辑表单 |
| 删除分组 | 点击删除按钮 | 二次确认后删除 |
| 调整排序 | 点击排序按钮 | 打开排序弹窗，拖拽调整分组顺序 |
| 管理速率倍率 | 点击速率倍率按钮 | 按模型组设置不同的费率倍率 |
| 管理 RPM 覆盖 | 点击 RPM 覆盖按钮 | 为特定模型设置独立的 RPM 限制 |

### 创建/编辑表单字段

| 字段 | 说明 |
|------|------|
| 名称 | 分组名称（必填） |
| 描述 | 分组描述（可选） |
| 从其他分组复制账号 | 多选下拉，复制已有分组的账号 |
| 速率倍率 | 默认费率倍率（必填，最小 0.001） |
| RPM 限制 | 每分钟请求限制（可选） |
| 独占模式 | 是否为独占分组（仅特定用户可用） |
| 订阅类型 | standard / subscription |
| 每日/每周/每月限额 | 订阅模式下的额度限制（美元） |
| 模型列表 | 支持的模型列表，可排序 |
| 图片生成定价 | 允许图片生成 + 独立费率 + 各分辨率价格 |
| 支持的模型范围 | claude / gemini_text / gemini_image |
| MCP XML 注入 | 启用/禁用 |
| Claude Code 限制 | 启用后限制 Claude Code 客户端 + 回退分组 |
| OpenAI Messages 调度 | 允许调度 + 家族映射 + 精确模型映射 |
| 账号过滤控制 | 仅 OAuth / 仅隐私集合 |
| 无效请求回退 | 回退分组选择 |
| 模型路由配置 | 启用路由 + 动态路由规则（模式匹配 + 账号选择） |

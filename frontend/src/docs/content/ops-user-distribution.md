# 分发模式用户页面

> 以下页面仅在分发模式（Distribution Mode）下可用。

---

## 兑换码 `/redeem`

使用兑换码获取余额、并发数或订阅。

### 兑换操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看当前余额 | 自动显示 | 梯度背景卡片展示余额和并发数 |
| 输入兑换码 | 在输入框中输入 | 支持字母数字混合码 |
| 提交兑换 | 点击兑换按钮 | 提交后显示兑换结果 |

### 兑换结果类型

| 类型 | 显示内容 |
|------|---------|
| balance | 添加金额 + 新余额 |
| concurrency | 添加的并发请求数 + 新并发数 |
| subscription | 分配的分组名 + 有效天数 |

### 近期活动

自动加载历史兑换和管理员调整记录，每条记录含类型图标、操作标题、时间、金额/天数。

---

## 我的订阅 `/subscriptions`

查看和管理所有订阅。

### 订阅卡片

每个订阅卡片显示：

| 区域 | 内容 |
|------|------|
| 头部 | 平台色彩圆点 + 分组名称 + 平台 badge + 分组描述 |
| 状态 | active（绿）/ expired（灰）/ cancelled（红） |
| 续费按钮 | 仅 active 状态显示，跳转到购买页面 |
| 过期时间 | 带倒计时天数，颜色编码 |
| 每日限额 | 进度条（绿→橙→红）+ 重置倒计时 |
| 每周限额 | 进度条 + 重置倒计时（168h 窗口） |
| 每月限额 | 进度条 + 重置倒计时（720h 窗口） |
| 无限额 | 显示 ∞ 图标 + "Unlimited" 提示 |

### 用户操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 浏览订阅 | 自动加载 | 网格布局展示所有订阅 |
| 立即续费 | 点击按钮 | 跳转到购买页面 |
| 查看用量 | 自动显示 | 各窗口的使用量和剩余配额 |

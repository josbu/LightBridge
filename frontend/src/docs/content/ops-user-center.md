# 用户中心页面（需登录）

本章介绍登录后用户可访问的所有功能页面。

---

## 用户仪表盘 `/dashboard`

用户主仪表盘，展示核心指标和使用概览。

| 功能区 | 功能 | 操作方式 |
|--------|------|---------|
| 核心指标卡片 | 查看余额、并发数、平台配额 | 自动加载 |
| 使用趋势图表 | 查看 Token 使用趋势 | 选择日期范围和粒度（天/小时） |
| 模型分布图表 | 查看各模型使用占比 | 自动加载 |
| 近期使用表格 | 查看最近 5 条使用记录 | 自动加载 |
| 快速操作面板 | 导航到常用功能 | 点击对应按钮 |
| 刷新数据 | 重新加载所有数据 | 点击刷新按钮 |

---

## API 密钥管理 `/keys`

API 密钥的创建、管理和监控。

### 列表操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索密钥 | 输入搜索关键词 | 按密钥名称模糊检索 |
| 按分组筛选 | 选择分组下拉 | 按 group_id 过滤 |
| 按状态筛选 | 选择状态下拉 | active / quota_exhausted / expired 等 |
| 创建密钥 | 点击「创建密钥」按钮 | 打开创建弹窗 |
| 编辑密钥 | 点击行内编辑按钮 | 打开编辑弹窗 |
| 删除密钥 | 点击删除按钮，确认删除 | 二次确认后删除 |
| 启用/禁用密钥 | 点击状态切换按钮 | 一键切换状态 |
| 复制密钥 | 点击复制按钮 | 复制完整密钥到剪贴板 |
| 使用密钥 | 点击终端图标按钮 | 展示 API 端点和示例代码 |
| 导入到 CC Switch | 点击上传图标按钮 | 通过 deeplink 导入到客户端 |
| 重置配额 | 点击重置按钮 | 重置已用配额 |
| 重置速率限制 | 点击重置按钮 | 重置各窗口的速率限制计数 |

### 创建/编辑表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | 文本输入 | 是 | 密钥名称 |
| 分组 | 可搜索下拉 | 是 | 选择密钥所属分组 |
| 自定义 API Key | 开关+文本 | 否 | 自定义密钥值（仅创建） |
| 启用 IP 限制 | 开关 | 否 | 启用后可配置 IP 白名单/黑名单 |
| 配额 | 数字输入 | 否 | 配额上限金额（美元），0 = 无限制 |
| 启用速率限制 | 开关 | 否 | 启用后可配置三窗口限额 |
| 5 小时/1 天/7 天限额 | 数字输入 | 否 | 各窗口限额 |
| 启用过期 | 开关 | 否 | 启用后可设置过期时间 |
| 过期时间 | 日期时间选择 | 否 | 快捷按钮：7/30/90 天 + 自定义 |

## 使用记录 `/usage`

查看 API 使用详情和费用统计。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看统计卡片 | 自动加载 | 总请求数、总 Token、总费用、平均时长 |
| 按密钥筛选 | 选择 API Key 下拉 | 按特定密钥过滤 |
| 按日期筛选 | 使用日期范围选择器 | 默认最近 7 天 |
| 导出 CSV | 点击导出按钮 | 自动分页拉取全量数据导出 |
| 查看费用详情 | 鼠标悬停费用列 | 弹出 tooltip 显示费用拆解 |
| 查看 Token 详情 | 鼠标悬停 Token 列 | 弹出 tooltip 显示 Token 明细 |
| 翻页浏览 | 使用分页控件 | 支持排序切换 |

### 表格列说明

| 列名 | 说明 |
|------|------|
| API Key | 关联的密钥名称 |
| Model | 使用的模型标识 |
| Endpoint | 入站端点 |
| Stream | 请求类型（WS / Stream / Sync） |
| Billing Mode | 计费模式（token / image） |
| Tokens | 输入/输出 Token 数 + 缓存 Token |
| Cost | 实际费用 |
| First Token | 首 Token 延迟（TTFT） |
| Duration | 请求总时长 |
| Created At | 请求时间 |

## 个人资料 `/profile`

管理个人信息、安全设置和第三方账号绑定。

| 功能区 | 功能 | 操作方式 |
|--------|------|---------|
| 基本信息 | 编辑个人资料字段 | 直接编辑并保存 |
| 修改密码 | 输入旧密码和新密码 | 提交修改 |
| 第三方账号绑定 | 绑定/解绑 OAuth 账号 | 点击绑定/解绑按钮 |
| 余额不足提醒 | 启用/禁用低余额通知 | 开关切换 |
| 设置提醒阈值 | 输入余额阈值 | 低于此值时发送通知 |
| 设置通知邮箱 | 添加额外通知邮箱 | 输入邮箱地址 |
| TOTP 双因素认证 | 启用/禁用 TOTP | 扫描二维码或输入密钥 |

支持的第三方账号绑定：LinuxDO、DingTalk、OIDC、WeChat（Web / Open平台 / 小程序）。

## 可用渠道 `/available-channels`

查看系统中可用的 API 渠道信息。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索渠道 | 输入搜索关键词 | 按渠道名、描述、平台名、分组名、模型名多维度过滤 |
| 刷新数据 | 点击刷新按钮 | 重新拉取渠道列表和用户专属费率 |
| 查看渠道详情 | 浏览表格 | 查看渠道的平台、分组和模型支持情况 |

---

## 渠道状态监控 `/monitor`

实时监控各渠道的健康状态。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看整体状态 | 自动加载 | 顶部横幅显示 operational / degraded |
| 切换时间窗口 | 选择时间窗口 | 支持 7d 等不同周期 |
| 手动刷新 | 点击刷新按钮 | 立即更新状态数据 |
| 开关自动刷新 | 切换自动刷新 | 刷新间隔可选 30 / 60 / 120 秒 |
| 查看渠道详情 | 点击渠道卡片 | 打开详情弹窗 |

---

## 自定义页面 `/custom/:id`

管理员配置的自定义内容页面。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| Markdown 渲染 | 自动 | 从 API 获取 Markdown 原文并渲染 |
| 目录导航 | 点击目录项 | 自动从 h1-h4 提取标题生成目录 |
| 展开/折叠目录 | 点击目录切换按钮 | 控制目录侧边栏显隐 |
| 代码块复制 | 点击复制按钮 | 每个代码块自动注入复制按钮 |
| iframe 嵌入 | 自动 | URL 模式下全屏 iframe 嵌入 |
| 新标签页打开 | 点击浮动按钮 | iframe 模式下在新标签页打开 |

# 管理员 - 分发模式专属

> 以下页面仅在分发模式（Distribution Mode）下可用。

---

## 订阅管理 `/admin/subscriptions`

管理用户订阅。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索订阅 | 输入搜索关键词 | 按用户或分组搜索 |
| 按状态筛选 | 选择状态下拉 | All / Active / Expired / Cancelled |
| 查看订阅详情 | 点击订阅行 | 展开订阅详细信息 |
| 编辑订阅 | 点击编辑按钮 | 修改订阅配置 |
| 取消订阅 | 点击取消按钮 | 取消用户订阅 |
| 为用户分配订阅 | 点击分配按钮 | 为指定用户分配订阅 |

---

## 公告管理 `/admin/announcements`

管理系统公告。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看公告列表 | 自动加载 | 展示所有公告 |
| 创建公告 | 点击创建按钮 | 打开创建表单 |
| 编辑公告 | 点击编辑按钮 | 打开编辑表单 |
| 删除公告 | 点击删除按钮 | 二次确认后删除 |
| 启用/禁用公告 | 点击状态切换 | 切换公告可见状态 |

### 公告字段

| 字段 | 说明 |
|------|------|
| 标题 | 公告标题 |
| 内容 | 公告正文（支持 Markdown） |
| 类型 | 信息 / 警告 / 错误 |
| 目标用户 | 所有用户 / 特定分组 |
| 有效期 | 公告显示的时间范围 |

---

## 兑换码管理 `/admin/redeem`

管理兑换码的生成和使用。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索兑换码 | 输入搜索关键词 | 按码值搜索 |
| 按类型筛选 | 选择类型下拉 | All / Balance / Concurrency / Subscription |
| 按状态筛选 | 选择状态下拉 | All / Unused / Used / Expired |
| 生成兑换码 | 点击生成按钮 | 打开生成表单 |
| 批量生成 | 选择批量生成 | 一次生成多个兑换码 |
| 删除兑换码 | 点击删除按钮 | 二次确认后删除 |
| 查看使用记录 | 点击兑换码 | 查看该码的使用历史 |
| 导出兑换码 | 点击导出按钮 | 导出为 CSV |

### 生成兑换码字段

| 字段 | 说明 |
|------|------|
| 兑换类型 | Balance（余额）/ Concurrency（并发）/ Subscription（订阅） |
| 余额金额 | Balance 类型的充值金额 |
| 并发数量 | Concurrency 类型的并发请求数 |
| 订阅分组 | Subscription 类型的分组选择 |
| 有效天数 | Subscription 类型的有效期 |
| 生成数量 | 一次生成的码数 |
| 有效期 | 兑换码的过期时间 |

---

## 优惠码管理 `/admin/promo-codes`

管理优惠码。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索优惠码 | 输入搜索关键词 | 按码值搜索 |
| 按状态筛选 | 选择状态下拉 | All / Active / Inactive / Expired |
| 创建优惠码 | 点击创建按钮 | 打开创建表单 |
| 编辑优惠码 | 点击编辑按钮 | 打开编辑表单 |
| 删除优惠码 | 点击删除按钮 | 二次确认后删除 |
| 启用/禁用 | 点击状态切换 | 切换优惠码可用状态 |
| 查看使用统计 | 点击优惠码 | 查看使用次数和金额统计 |

### 优惠码字段

| 字段 | 说明 |
|------|------|
| 码值 | 优惠码字符串 |
| 折扣类型 | 固定金额 / 百分比 |
| 折扣值 | 折扣金额或百分比 |
| 最小充值金额 | 使用优惠码的最低充值额 |
| 最大使用次数 | 优惠码可使用的总次数 |
| 有效期 | 优惠码的过期时间 |
| 适用分组 | 限制使用的订阅分组 |

## 风控管理 `/admin/risk-control`

配置和管理风控规则。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 查看风控规则 | 自动加载 | 展示所有风控规则 |
| 创建规则 | 点击创建按钮 | 打开创建表单 |
| 编辑规则 | 点击编辑按钮 | 打开编辑表单 |
| 删除规则 | 点击删除按钮 | 二次确认后删除 |
| 启用/禁用规则 | 点击状态切换 | 切换规则生效状态 |
| 调整规则优先级 | 拖拽排序 | 调整规则执行顺序 |
| 测试规则 | 点击测试按钮 | 使用测试数据验证规则 |

### 风控规则字段

| 字段 | 说明 |
|------|------|
| 规则名称 | 风控规则名称 |
| 规则类型 | IP 限制 / 频率限制 / 金额限制 / 行为分析 |
| 触发条件 | 规则触发的条件 |
| 处置动作 | 警告 / 限制 / 封禁 |
| 生效范围 | 全局 / 特定分组 / 特定用户 |
| 有效期 | 规则生效的时间范围 |

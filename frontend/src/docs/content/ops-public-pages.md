# 公共页面（无需登录）

本章介绍所有无需登录即可访问的页面功能。

---

## 首页 `/home`

系统首页，支持管理员自定义内容或默认落地页。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 自定义内容展示 | 自动 | 管理员配置 `home_content` 后，以 iframe 或内嵌 HTML 渲染 |
| 默认落地页 | 自动 | 未配置时展示英雄区 + 特色卡片 + AI 提供商展示 |
| 导航到登录 | 点击「登录」按钮 | 未登录状态显示 |
| 导航到控制台 | 点击「控制台」按钮 | 已登录时显示，管理员跳管理仪表盘，普通用户跳用户仪表盘 |
| 切换语言 | 点击语言切换按钮 | 支持中文/英文 |
| 切换主题 | 点击主题切换按钮 | 亮色/暗色模式切换 |

---

## 登录 `/login`

用户登录页面，支持多种登录方式。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 邮箱密码登录 | 输入邮箱和密码，点击「登录」 | 基础登录方式 |
| 显示/隐藏密码 | 点击密码框右侧眼睛图标 | 切换密码可见性 |
| 人机验证 | 自动触发 | 启用 Cloudflare Turnstile 时需通过验证 |
| 登录协议确认 | 勾选同意框或弹窗确认 | 根据管理员配置 |
| OAuth 快捷登录 | 点击对应提供商按钮 | 支持 GitHub、Google、LinuxDo、钉钉、微信、OIDC |
| TOTP 二次验证 | 输入 6 位验证码 | 启用 TOTP 的用户登录后弹出验证弹窗 |
| 跳转到注册 | 点击「注册」链接 | 底部注册入口 |
| 忘记密码 | 点击「忘记密码」链接 | 需管理员启用密码重置功能 |

### OAuth 登录流程

1. 点击 OAuth 提供商按钮（如 GitHub）
2. 跳转到提供商授权页面
3. 授权后回调到 `/auth/callback`
4. 系统自动处理：已有账号直接登录 / 需邀请码则输入后创建 / 需创建账号则填写密码 / 需绑定已有账号则输入邮箱密码 / 需 TOTP 验证则输入验证码

## 注册 `/register`

新用户注册页面。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 邮箱注册 | 输入邮箱，点击「创建账户」 | 基础注册方式 |
| 密码设置 | 输入密码和确认密码 | 需符合安全要求 |
| 邀请码（可选） | 输入邀请码 | 启用邀请码时为必填项 |
| 优惠码（可选） | 输入优惠码 | 带实时校验，显示赠送金额 |
| OAuth 快捷注册 | 点击提供商按钮 | 携带 affiliate code 注册 |
| 自动预填 | URL 参数 `?aff=`、`?promo=` | 自动填充邀请码或优惠码 |

---

## 邮箱验证 `/email-verify`

邮箱验证码输入页面。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 输入验证码 | 输入 6 位数字 | 自动从 sessionStorage 读取注册数据 |
| 验证并创建账户 | 点击按钮提交 | 完成注册流程 |
| 重新发送验证码 | 点击重发按钮 | 带倒计时 |

---

## 忘记密码 `/forgot-password`

请求密码重置邮件。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 提交重置请求 | 输入邮箱，点击提交 | 后端发送重置链接到邮箱 |
| 返回登录 | 点击链接 | 提交成功后显示 |

---

## 重置密码 `/reset-password`

实际设置新密码。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 设置新密码 | 输入新密码和确认密码 | 邮箱从 URL 参数自动填充且只读 |
| 提交重置 | 点击按钮 | 三种状态：表单 / 成功提示 / 令牌无效警告 |

## 系统初始化向导 `/setup`

首次安装时的四步配置向导。

| 步骤 | 功能 | 操作方式 |
|------|------|---------|
| 步骤 1：数据库配置 | 配置 PostgreSQL 连接 | 填写 host/port/user/password/dbname/SSL，点击「测试连接」 |
| 步骤 2：Redis 配置 | 配置 Redis 连接 | 填写 host/password/db/TLS，点击「测试连接」 |
| 步骤 3：管理员账户 | 创建管理员账号 | 填写邮箱/密码，选择部署模式（分发/个人） |
| 步骤 4：确认安装 | 确认配置并安装 | 查看摘要，点击「安装」，等待重启 |

已完成初始化时访问 `/setup` 会自动重定向到仪表盘。
